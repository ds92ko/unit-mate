## 👀 Introduction

[UnitMate](https://unit-mate.vercel.app/)는 CSS 단위 계산기 토이 프로젝트입니다.<br/>
주요 목적은 단위 간 변환과 기본 계산을 편리하게 수행하여 빠르고 정확한 CSS 작업을 지원하는 것입니다.

### 🌟 Key Features

- 일반 계산기
- PX ↔️ 퍼센트 단위 변환
- PX ↔️ VW | VH 단위 변환
- PX ↔️ EM | REM 단위 변환

### ✨ Additional Features

- 계산 히스토리 및 북마크
- 다크/라이트 모드 + 컬러 테마

## 🪄 Getting Started

### ⚙️ Setup

```bash
$ git clone https://github.com/ds92ko/unit-mate.git
$ cd unit-mate
```

### 📦 Installation

```bash
$ npm install
```

### 🚀 Usage

```bash
$ npm start
```

### 🌐 Deployment

```bash
npm run build
```
